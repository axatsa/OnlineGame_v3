## Подзадачи

### 5.1 Inline-редактор результата

**Файлы:** `front/src/components/ResultEditor/`, применяется в каждом генераторе

**Что делать:**
- После генерации показывать результат в режиме просмотра
- Кнопка «Редактировать» → переключает в edit-режим
- В edit-режиме каждый вопрос/ответ — это `contenteditable` или `<textarea>`
- Кнопки: ✅ Сохранить, ❌ Отменить изменения, ➕ Добавить вопрос, 🗑 Удалить вопрос

**Пример для Quiz:**
```tsx
// Интерактивный список вопросов
questions.map((q, i) => (
  <QuestionCard key={i} question={q} onEdit={(updated) => updateQuestion(i, updated)} />
))
```

- После редактирования → «Скачать DOCX» использует отредактированные данные
- Сохранять отредактированный вариант в `generation_logs` (поле `edited_result`)

### 5.2 Шаблоны заданий

**Файлы:** `backend/apps/generator/models.py` и `router.py`, `front/src/pages/templates/Templates.tsx`

**Модель:**
```python
class Template(Base):
    __tablename__ = "templates"
    
    id: int
    user_id: int | None      # NULL = системный шаблон
    feature: str             # "quiz", "math", "crossword" etc.
    name: str                # "Математика 3 класс — стандарт"
    description: str
    params: JSON             # те же поля, что в форме генератора
    is_system: bool = False  # системные показываются всем
    created_at: datetime
```

**API:**
```
GET  /api/v1/templates?feature=quiz          → список шаблонов
POST /api/v1/templates                       → сохранить текущие параметры как шаблон
POST /api/v1/templates/{id}/apply            → заполнить форму параметрами шаблона
DELETE /api/v1/templates/{id}               → удалить (только свои)
```

**Системные шаблоны** (добавить через seed):
- Квиз: «Биология 9 класс — ОГЭ», «История 11 класс — ЕГЭ»
- Математика: «3 класс — Дроби», «5 класс — Уравнения»
- Кроссворд: «Животные», «Страны мира»

**Frontend:**
- В каждой форме генератора добавить кнопку «Загрузить шаблон» → выпадающий список
- Кнопка «Сохранить как шаблон» → модал с вводом названия
