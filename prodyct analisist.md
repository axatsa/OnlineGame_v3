# ClassPlay — Анализ продукта и роадмап для B2C / B2B продаж

> Дата: 30 марта 2026  
> Цель: запуск продаж учителям напрямую (B2C) и пакетные лицензии школам / репетиторским центрам (B2B)

---

## 1. Что уже реализовано ✅

### 🔐 Аутентификация и роли
- JWT авторизация (email + password)
- Две роли: `teacher` и `super_admin`
- Защищённые роуты на фронте и бэке
- Смена пароля из профиля
- Auto-redirect при истечении токена

### 🏫 Управление классами
- CRUD классов (название, год обучения, кол-во учеников)
- AI-контекст класса — описание для персонализации генерации
- Уровни 1–11 класс с автоподсказками по теме
- Активный класс передаётся во все генераторы
- Хранение в PostgreSQL, API через FastAPI

### 🤖 AI-генераторы (OpenAI GPT)
- **Математические задания** — тема + сложность + кол-во, 10–30 задач, экспорт DOCX
- **Кроссворд** — генерация слов + автораскладка на сетке, экспорт DOCX
- **Викторина (Quiz)** — 4 варианта ответа, лист ответов, экспорт DOCX
- **Задание (Assignment)** — полный тест с условием и ответами, print-view A4
- Поддержка языков: Русский / O'zbek
- Передача контекста класса в каждый запрос

### 🎮 Игры (Smartboard)
- **Tug of War** — командная битва на вопросах, фоновая музыка, секундомер, разные вопросы для команд
- **Jeopardy** — 5 категорий × 5 вопросов, AI-генерация, командные очки
- **Memory Matrix** — карточки-пары, 4 темы, 4 уровня сложности
- **Balance Scales** — математические весы, уравновесить задачами
- **Philword (Word Search)** — поиск слов в сетке, 3 уровня сложности
- **Crossword** — интерактивный кроссворд в браузере с проверкой ответов
- Единый `GameShell` с кнопками Help / Restart / Volume

### 📚 Детская библиотека
- Генерация 10-страничных детских книжек (Gemini 2.0)
- 10 AI-иллюстраций параллельно (asyncio.gather)
- Book Reader с постраничной навигацией (обложка → текст → иллюстрация)
- Жанры, возрастные группы, 3 языка
- Удаление книг, список в таблице

### 🛠 Инструменты учителя
- **Рулетка** — случайный выбор ученика, canvas-анимация
- **Доска для рисования** — brush colors, sizes, ластик, очистка
- **Генератор заданий** — быстрый вариант с textarea и print-view

### 👨‍💼 Панель суперадмина
- Разделы: Dashboard, Teachers, Organizations, AI Monitor, Finances, System
- Метрики: MRR, ARR, токены, расходы
- Таблица учителей с блокировкой / сбросом пароля
- Организации с лицензиями и сроком действия
- История платежей
- AI-провайдер switch (Gemini / OpenAI)
- Системные объявления
- Журнал аудита
- Экспорт CSV + DOCX для всех секций

### ⚙️ Инфраструктура
- Docker + Docker Compose (dev + prod)
- Nginx reverse proxy + SPA routing
- PostgreSQL с миграциями через SQLAlchemy
- Token usage tracking по фичам
- Seed-скрипты для начальных данных
- React Query для кэширования данных
- Мультиязычность RU / UZ через собственный контекст

## 5. Ключевые технические решения

### Тарифы — рекомендуемая схема
```python
# models.py
class Subscription(Base):
    user_id: int
    plan: str          # "free" | "pro" | "school"
    status: str        # "active" | "cancelled" | "past_due"
    current_period_end: datetime
    stripe_subscription_id: str
    generations_used: int   # сбрасывается каждый месяц
    generations_limit: int  # -1 = unlimited
```

### Лимиты — middleware
```python
# Добавить в каждый /generate/* роут
def check_quota(user: User, db: Session):
    sub = db.query(Subscription).filter_by(user_id=user.id).first()
    if not sub or sub.status != "active":
        raise HTTPException(402, "Upgrade required")
    if sub.generations_limit != -1 and sub.generations_used >= sub.generations_limit:
        raise HTTPException(429, "Monthly limit reached")
    sub.generations_used += 1
    db.commit()
```

### Регистрация — новые роуты
```
POST /api/auth/register     — создание аккаунта
POST /api/auth/verify-email — подтверждение токена из письма
POST /api/auth/forgot       — отправить reset email
POST /api/auth/reset        — сменить пароль по токену
```

## 6. Монетизация — ориентиры рынка

| Сегмент | Модель | Цена | LTV |
|---------|--------|------|-----|
| Частный учитель | Monthly subscription | $8–12/мес | $96–144 |
| Репетитор (5–10 учеников) | Pro plan | $12–18/мес | $150–220 |
| Частная школа (до 20 учителей) | School plan | $59–99/мес | $700–1200/год |
| Государственная школа | Annual contract | $500–2000/год | переговорный |
| Репетиторский центр | Per-seat | $5–8/учитель/мес | зависит от размера |

**Рекомендуемый старт:** Free (10 генераций) → Pro $9/мес → School $49/мес за 10 мест

## 7. Что НЕ делать сейчас

- ❌ **Мобильное приложение** — сначала web, потом native
- ❌ **Собственная LLM** — дорого, нет смысла на этом этапе
