## Контекст

Для продажи лицензий школам нужны инструменты управления: директор должен видеть активность учителей, IT-отдел должен иметь возможность массово добавить пользователей, суперадмин — генерировать invite-ссылки.

## Подзадачи

### 8.1 Школьный дашборд (для суперадмина)

**Файлы:** `front/src/pages/admin/OrganizationDetail.tsx`, `backend/app/routers/admin_orgs.py`

**Что делать:**

**Backend — новый эндпоинт:**
```
GET /api/admin/organizations/{org_id}/stats
```
Ответ:
```json
{
  "org_name": "Школа №15",
  "total_teachers": 12,
  "active_last_7_days": 8,
  "total_generations": 340,
  "teachers": [
    { "name": "Иванова А.А.", "generations_30d": 45, "last_active": "2026-03-29" },
    ...
  ]
}
```

**Frontend:**
- Страница организации в AdminPanel → вкладка «Активность»
- Таблица учителей с их статистикой
- Общие метрики: всего учителей, активных, генераций за месяц

### 8.2 Invite-ссылка для школы *(зависит от email)*

**⚠️ Реализовать после внедрения email-верификации**

**Логика (для проектирования):**
- Суперадмин → страница организации → «Создать invite-ссылку»
- Backend генерирует токен и сохраняет:
  ```python
  class InviteToken(Base):
      token: str (UUID)
      org_id: int
      expires_at: datetime   # +7 дней
      max_uses: int          # например 30
      uses_count: int = 0
  ```
- Ссылка: `https://classplay.app/join/{token}`
- При регистрации по ссылке → пользователь автоматически привязывается к org

### 8.3 Bulk import учителей (CSV)

**Файл:** `backend/app/routers/admin_users.py`, `front/src/pages/admin/BulkImport.tsx`

**Что делать:**

**Формат CSV:**
```csv
name,email,role
Иванова Анна,ivanova@school15.ru,teacher
Петров Сергей,petrov@school15.ru,teacher
```

**Backend:**
```
POST /api/admin/organizations/{org_id}/import-csv
```
- Принимает multipart файл
- Парсит CSV (библиотека `csv` или `pandas`)
- Создаёт пользователей с временным паролем (UUID, первые 8 символов)
- Возвращает список созданных: `[{email, temp_password}]`
- Если пользователь с email уже существует — пропустить, добавить в список ошибок

**Frontend:**
- Страница организации → вкладка «Загрузить учителей»
- Drag & drop CSV файл
- Preview: таблица с именами/email перед импортом
- После импорта: показать таблицу результатов (создан/пропущен/ошибка)
- Кнопка «Скачать как XLSX» список пар email + временный пароль
