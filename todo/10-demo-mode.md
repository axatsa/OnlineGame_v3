## Подзадачи

### 10.1 Demo-режим без регистрации

**Файлы:** `backend/app/routers/demo.py`, `front/src/pages/Demo.tsx`

**Логика:**
- Отдельный эндпоинт без авторизации: `POST /api/demo/generate`
- Только один генератор — Quiz (самый простой для демо)
- Ограничение по IP: **3 запроса в час** (через `slowapi`)
- Не сохраняется в историю, не использует квоту пользователя
- Отдельный счётчик токенов для demo (budget_demo_tokens в config)

**Frontend — страница `/demo`:**
- Минимальная форма: тема + язык + количество вопросов (фиксировано 5)
- После генерации — показать Quiz
- Баннер снизу: «Понравилось? Зарегистрируйтесь бесплатно →»
- Кнопка «Скачать DOCX» → редирект на регистрацию

**Лендинг:**
- Кнопка «Попробовать бесплатно» → `/demo`
